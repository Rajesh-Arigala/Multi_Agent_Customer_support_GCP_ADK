# Data Privacy Notes

## Purpose
This document defines privacy expectations for stakeholder and opportunity data.

## Data Collected
```text
name
organization
role
email
phone
country
engagement type
intent summary
preferred contact channel
```

## Principles
```text
collect only useful professional contact details
avoid storing private personal information
do not expose stakeholder details publicly
keep staff/admin access controlled
log opportunity actions responsibly
```

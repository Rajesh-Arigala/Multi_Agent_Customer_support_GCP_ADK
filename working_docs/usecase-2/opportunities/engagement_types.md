# Engagement Types

## Purpose
This document defines the opportunity types RAG by RABBIT should classify and capture.

## Engagement Type List
```text
Hiring / Job Opportunity
Consulting Opportunity
Fractional Leadership / Advisor Role
Business-AI Transformation Discussion
MLOps / AI Platform Project Discussion
GenAI / RAG Implementation Discussion
Workflow Agent / Action Agent Discussion
Startup / Founder Collaboration
Speaking Engagement
Workshop / Training Session
Guest Lecture / Panel Discussion
Presentation To Third Parties
Investor / Capital Conversation
Partnership / Strategic Alliance
Media / Podcast / Interview Request
Mentorship / Professional Networking
Demo Request For RABBIT Or Projects
Referral / Introduction Request
General Contact
```

## Example Classification
```text
User: Can Rajesh present RABBIT to our leadership team?
Engagement type: Presentation To Third Parties
Capital type: social_capital + possible money_capital
Priority: medium or high depending on audience and organization
```

## Required Capture Fields
```text
name
organization
role
contact_channel
engagement_type
capital_type
intent_summary
timeline
priority
next_action
```

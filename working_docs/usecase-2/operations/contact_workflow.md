# Contact Workflow

## Purpose
This document defines how RAG by RABBIT moves from conversation to contact action.

## Contact Channels
```text
email
phone / WhatsApp
LinkedIn
GitHub
meeting request
demo request
```

## Workflow
```text
Stakeholder shows intent
-> collect name, organization, role, contact
-> classify engagement type
-> create engagement ticket
-> prepare contact summary
-> route to Rajesh
```

## Example
User:

```text
Can we schedule a discussion about a consulting project?
```

Agent:

```text
Collects organization, problem, contact details, and creates Consulting Opportunity ticket.
```

## Boundary
Do not promise immediate live availability unless calendar integration exists.

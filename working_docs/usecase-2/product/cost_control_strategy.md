# Cost Control Strategy

## Purpose
This document defines how RAG by RABBIT stays economically efficient.

## Cost Principles
```text
reuse existing RABBIT RAG corpus
use retrieval before generation
avoid web search unless needed
use structured forms for opportunity capture
use cheaper model routes for simple actions
log cost per opportunity
start with Google Sheets before CRM/database
```

## Business Metrics
```text
cost per conversation
cost per qualified opportunity
cost per money-capital lead
cost per social-capital lead
engagement conversion rate
high-value escalation count
```

# User Journey Examples

## Purpose
This document gives realistic examples for RAG by RABBIT.

## Journey 1: Recruiter Evaluation
```text
User: Is Rajesh suitable for an AI transformation role?
Agent: Answers from website/project evidence, explains business-tech fit, and offers to capture the role details.
Outcome: Hiring / Job Opportunity ticket.
```

## Journey 2: Consulting Lead
```text
User: We need help building a RAG assistant for our company.
Agent: Explains relevant proof from RABBIT and Rajesh's projects, then collects company, problem, timeline, and contact details.
Outcome: Consulting Opportunity ticket.
```

## Journey 3: Speaking Engagement
```text
User: Can Rajesh speak at our AI meetup?
Agent: Collects event name, audience, topic, date, format, and organizer contact.
Outcome: Speaking Engagement ticket.
```

## Journey 4: Third-Party Presentation
```text
User: Can Rajesh present Sentinel to our leadership team?
Agent: Collects organization, audience profile, presentation goal, date window, and decision-maker contact.
Outcome: Presentation To Third Parties ticket.
```

## Journey 5: Proof-Of-Work Question
```text
User: Did Rajesh build this agent himself?
Agent: Explains that RAG by RABBIT is a live proof-of-work system built by Rajesh, while staying factual and professional.
Outcome: Increased credibility.
```

## Journey 6: Off-Scope Personal Question
```text
User: Tell me private family details.
Agent: Refuses politely and redirects to Rajesh's professional profile, projects, role fit, or contact path.
Outcome: Privacy boundary preserved.
```

# Usecase 2 Working Documentation

## Purpose
This folder contains the working documentation for RAG by RABBIT, expanded as RABBIT Agent GALA. It defines the product, architecture, data, operations, safety, economics, and phased build plan before implementation.

## Use Case Summary
RAG by RABBIT is an action-oriented professional opportunity agent for `rajesharigala.com`. It answers stakeholder questions from Rajesh Arigala's website and project evidence, makes clear that the agent itself was built by Rajesh, qualifies professional intent, and routes national and international money-capital and social-capital opportunities.

## Folder Map
```text
profile/        Rajesh positioning, proof-of-work story, public contact boundaries
opportunities/  engagement types, money capital, social capital
use_cases/      main use case and user journey examples
architecture/   system architecture, agent schema, data, retrieval, deployment
operations/     engagement ticket lifecycle, lead routing, contact workflow
safety/         privacy, professional boundaries, anti-hallucination rules
product/        cost control and business value
one-page/       business and implementation summaries
phases/         phase-by-phase build plan with visuals
```

## Core Business Intent
```text
Answer with evidence -> prove capability -> qualify stakeholder intent -> capture opportunity -> route to Rajesh.
```

## Implementation Boundary
These documents define intent. They are not implementation. Implementation starts only after explicit approval.

# Data Privacy Notes

## Purpose
This document defines privacy expectations for early production design.

## Data Collected
Level 1:

```text
name
phone
preferred_language
```

Level 2:

```text
appointment preferences
service interest
reason for visit
consultation type
```

## Principles
```text
collect minimum necessary data
do not expose private data in public responses
keep staff access controlled through Google Sheets permissions
avoid storing raw audio unless explicitly required
log actions but not unnecessary medical details
```

## Open Questions
- Consent wording.
- Retention period.
- Who can access Google Sheets.
- Whether audit logs contain message summaries or full text.

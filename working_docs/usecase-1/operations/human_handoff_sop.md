# Human Handoff SOP

## Purpose
This document defines when humans must take over.

## Handoff Cases
```text
urgent or severe medical concern
appointment confirmation
complex medical history
pricing or payment questions if not approved in FAQ
angry/frustrated user
low-confidence answer with high user intent
```

## Minimum Details
```text
name
phone
message summary
preferred language
reason for handoff
urgency level
```

## Desk Action
```text
review ticket
call user
update status
add notes
close or escalate further
```
